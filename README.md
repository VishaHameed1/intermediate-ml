# 🚀 AI/ML Intermediate Projects Portfolio

This repository documents my transition from foundational Python to **Intermediate Level Machine Learning and Deep Learning**. It serves as a comprehensive collection of projects built using **Python**, **PyTorch**, **scikit-learn**, and real-world datasets.

---

## 🛠️ Project Roadmap & Overview

| Project | Category | Key Tech | Description |
| --- | --- | --- | --- |
| **🖼️ CNN Image Classifier** | Deep Learning | PyTorch / CNN | Classifies Cats vs. Dogs using Data Augmentation for generalization. |
| **📉 Customer Churn** | Classification | Scikit-learn | Predicts user retention patterns using feature scaling and EDA. |
| **👥 Customer Segmentation** | Clustering | KMeans | Unsupervised learning to group purchasing behaviors. |
| **💳 Fraud Detection** | ML / Security | Random Forest | High-precision detection of fraudulent transactions in `creditcard.csv`. |
| **💰 Loan Default Prediction** | Predictive ML | XGBoost | applicant risk assessment and financial feature engineering. |
| **🛒 Market Basket Analysis** | Data Mining | Apriori | Association rule mining to find hidden product purchase patterns. |
| **🎬 Movie Recommender Pro** | NLP / RecSys | Collaborative Filtering | User-rating based engine using the MovieLens dataset. |
| **📈 Stock Price Prediction** | Time Series | LSTM / Regression | Sequential data forecasting for financial market trends. |
